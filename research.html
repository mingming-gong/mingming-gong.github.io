Research (Selected Topics)

Causal Discovery

Causal discovery is the problem of inferring the causal relationships from observational data. It is a fundamental task in various disciplines of science and engineering. We are working on computational methods to discover causal relationships from real-world complex data, with theoretical guarantees.

  • Causal discovery from imperfectly sampled time series data [ICML'15][ICML'15][UAI'17][NeurIPS'19]

  • Causal discovery from nonstationary data [ICML'19]

  • Causal discovery from data with measurement error [UAI'18]

  • Causal discovery from data with missing values [AAAI'20][NeurIPS'22]


Causal Machine Learning

Current machine learning is mostly driven by statistical correlations. However, relying on pure statisitical correlations makes machine learning models hard to generalize to new domains and tasks. Some argue that this requires learning not only the statistical correlations, but also the causal generative process behind the data. We are exploring causal systems' properties and how they could benefit knowledge transfer in machine learning. In addition, we are interested in improving the fairness, robustness, and interpretability of machine learning models from a causal perspective.

  • Domain adaptation/generalization [AAAI'15][ICML'16][AAAI'18][ECCV'18] [AISTATS'19][ICML'20][ICML'20] [NeurIPS'20][NeurIPS'20][NeurIPS'21]

  • Fairness in machine learning [CLeaR'22][NeurIPS'22]

  • Robust machine learning [NeurIPS'21][ICLR'22]

  • Generalization in reinforcement learning [ICLR'22]

  • Unsupervised image-to-image translation [ICCV'21]


3D Computer Vision

3D computer vision aims to model and understand the visual world by inferring 3D structures from 2D images. We are interested in deep learning methods for depth estimation, 3D reconstruction, 3D model retrieval, etc. In addition, we are leveraging the causal process from the 3D world to 2D images to improve the generalization of 3D reconstruction models.

  • Depth estimation [CVPR'18][CVPR'19][ECCV'22]

  • Neural radiance fields [ECCV'22]

  • 3D model retrieval [NeurIPS'20]

  • Interest points [IJCV'22]

  • Benchmark dataset [IJCV'21]
